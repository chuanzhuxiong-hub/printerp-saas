# PrintERP 第一版商用验收报告

验收日期：2026-06-10

验收基准：`PrintERP_Commercial_PRD_Optimized_v2.md` 第一阶段范围。

## 结论

PrintERP 第一版代码与本地部署环境已完成商用发布验收，自动化验收结果为通过。

第二、第三阶段明确规划的外部合规 API、定时竞品采集、真实 AI 图片生成和自动发布不计入第一版阻断项。

## 已完成范围

- 订单、采购、库存、生产、成本、发货、售后、报表与数据导入闭环
- 可编辑采购日期、kg/g 自动换算、采购单编辑与作废
- 打印机配件更换成本、工具设备与折旧成本
- 打印机配件纳入统一库存主表与统一库存流水
- 售后补发按订单 SKU 与 BOM 自动扣减成品和包装库存
- 采购修改使用追加更正流水，保留原始采购入库记录
- 数据初始化、权限控制、审计日志、租户隔离与禁用员工会话即时失效
- 产品中心集成 SKU、BOM、3D 模型版本、打印参数与当前生产版本
- 产品中心内的 AI 标题、主图、详情页工作流
- 产品中心内的竞品监控与自动选品池
- 每个产品最多绑定 5 个竞品
- 竞品销量区分 `salesDisplayValue`、`salesEstimate`、`salesActual`
- 第一版手动录入、表格导入与半自动分析
- 使用帮助覆盖全部菜单
- 数据库备份、恢复与恢复演练

## 自动化验收

执行命令：

```powershell
npm run test:acceptance
```

结果：

- 生产构建通过，共 102 个页面与 API 路由
- 冒烟测试通过：51 项
- 帮助覆盖通过：37 个菜单、39 个帮助主题
- 业务、权限、租户隔离、登录限流与依赖安全门禁全部通过
- 性能测试通过：120 请求，并发 12，p50 109.7ms，p95 377.2ms，最大 467.7ms
- 数据库备份恢复演练通过：56 张业务表、9 个迁移
- 生产依赖安全门禁：0 critical、0 high、0 moderate
